Practica 1 - Web services

Crear un usuario:
POST .../rest/usuario
JSON: { alias: 'alias', name: 'name', surname: 'surname' }

Lista de usuarios:
GET .../rest/usuario
JSON: -

Datos de un usuario:
GET .../rest/usuario/:id
JSON: -

Borrar un usuario:
DELETE .../rest/usuario/:id
JSON: -

Setear un propiedad de un usuario:
PUT .../rest/usuario/:id/propiedad
Propiedades posibles: name, surname, age, phone, group, photo
JSON: { value: 'value'}

Leer un propiedad de un usuario:
GET .../rest/usuario/:id/propiedad
Propiedades posibles: name, surname, age, phone, group, photo
JSON: -

Limpiar un propiedad:
DELETE .../rest/usuario/:id/propiedad
Propiedades posibles: name, surname, age, phone, group, photo
JSON: -

Lista de grupos:
GET .../rest/grupo
JSON: -

Crear un grupo:
POST .../rest/grupo
JSON: { name: 'groupname' }
